== Description ==
سیستم ماژولار مدیریت تامین‌کنندگان و تولید هوشمند SKU برای فروشگاه لپ‌ایکس.

== Installation ==
1. افزونه را دانلود کنید.
2. در وردپرس به بخش افزونه‌ها > افزودن بروید.
3. فایل zip را آپلود و نصب کنید.
4. افزونه را فعال کنید.
